Hands-on UI/UX Assessment:

The intent of this execerise is to show your approach, abilities, code organization, and solution to a common UI element found in web applications

Acceptance Criteria:

The requests is to build a responsive dismissible banner using HTML, CSS, and JS with the assets provided below. Banner design provided below.

  1. Using HTML and CSS, match the given design as close as possible. *Not asking for pixel perfect.
  2. The layout of the banner should extend across the entire width of the browser window.
  3. The banner should be responsive for small and large viewports.
  4. Events:
    1. Add click event on the close button to dismiss banner.
    2. Set a cookie when the banner is dismissed.
    3. Console log cookie name, value and expiration date.
    4. Use the following Cookie Values:
      1. Name: covidBanner
      2. Value: True
      3. Expires in 2 days
      Console log result: covidBanner, true, Expiration date
  5. The banner should be reusable, have multiple banners on the same page with it's dismiss function independent of each other. *You can copy/paste the same banner, the point is to dismiss them one by one.
  6. Add call to actions that opens a new browser window or tab. *use # for href placeholder.
  7. Fork this CodePen and show your work below.
